# E-commerce Sales Analytics (SQL + Power BI)

## Goal
Analyze e-commerce sales data to answer key business questions (revenue trends, top products, customer behavior) and build an interactive Power BI dashboard.

## Tools
- SQL (MySQL / PostgreSQL)
- Power BI
- Excel or Python (optional for cleaning)

## Project Structure
- data/ : raw & cleaned data (or links)
- sql/ : queries and analysis scripts
- powerbi/ : dashboard file + screenshots
- docs/ : data dictionary, notes, screenshots
- notebooks/ : optional Python EDA/cleaning

## Plan (4 Weeks)
- Week 1: Dataset + cleaning + load into SQL
- Week 2: SQL analysis + KPI queries
- Week 3: Power BI dashboard
- Week 4: Insights + documentation + resume bullets

## Key Questions (draft)
- What are monthly sales trends?
- Which products/categories generate the most revenue?
- Who are the top customers?
- What is the average order value (AOV)?
- Repeat customer rate over time?
